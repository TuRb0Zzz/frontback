# Отчёт по третьей контрольной работе


## Практическое занятие №13:

В ходе занятия была реализована регистрация Service Worker для кэширования статических ресурсов и обеспечения офлайн-доступа. Заметки сохраняются в localStorage и остаются доступными даже при отсутствии интернета. Service Worker перехватывает сетевые запросы и возвращает ресурсы из кэша, что позволяет приложению загружаться при отключённой сети.

---

## Практическое занятие №14:

Был добавлен файл manifest.json и иконки разных размеров для возможности установки приложения на рабочий стол. Настроен favicon для отображения иконки во вкладке браузера, что позволяет идентифицировать приложение среди других вкладок. Также были добавлены мета-теги для улучшения отображения на мобильных устройствах.

---

## Практическое занятие №15:

Был настроен локальный HTTPS сервер с помощью утилиты mkcert, что позволило использовать Service Worker в безопасном режиме. Реализована архитектура App Shell, при которой каркас приложения кэшируется и загружается мгновенно, а контент подгружается динамически. Навигация между страницами "Главная" и "О приложении" происходит без перезагрузки страницы.

---

## Практическое занятие №16:

Был создан Node.js сервер с использованием Socket.IO для обмена сообщениями в реальном времени между клиентами. Реализованы push-уведомления через VAPID-ключи, что позволяет отправлять уведомления даже при закрытом или свёрнутом браузере. Добавлены кнопки для включения и отключения уведомлений, а также управление подписками через эндпоинты /subscribe и /unsubscribe

---

## Практическое занятие №17:

В приложение была добавлена форма для создания заметок с напоминанием, где пользователь может указать текст и дату с временем. При нажатии на кнопку "Отложить на 5 минут" в push-уведомлении, время напоминания в localStorage увеличивается на 5 минут без участия сервера. Под каждой заметкой с напоминанием отображается иконка и установленное время, а сама заметка сохраняется в localStorage с уникальным идентификатором.
