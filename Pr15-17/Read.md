# Отчёт по второй контрольной работе


## Практическое занятие №13:

Реализована регистрация Service Worker для кэширования статических ресурсов и обеспечения офлайн-доступа. Заметки сохраняются в localStorage и доступны без интернета.

---

## Практическое занятие №14:

Добавлен manifest.json и иконки разных размеров для установки приложения на рабочий стол. Настроен favicon для отображения иконки во вкладке браузера.

---

## Практическое занятие №15:

Настроен локальный HTTPS сервер через mkcert. Реализована архитектура App Shell с динамической загрузкой контента и навигацией между страницами.

---

## Практическое занятие №16:

Создан Node.js сервер с Socket.IO для обмена сообщениями в реальном времени. Реализованы push-уведомления через VAPID-ключи и управление подписками.

---

## Практическое занятие №17:

Добавлена форма для создания заметок с напоминанием (текст + дата/время). Реализована кнопка "Отложить на 5 минут" с обновлением времени в localStorage.
